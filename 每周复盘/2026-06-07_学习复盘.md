### 1. 获取正文

网页链接场景下，微信公众号文章通常可以用 `urllib.request` 读取：

```python
import urllib.request
url = '<用户链接>'
req = urllib.request.Request(url, headers={
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 Chrome/121 Safari/537.36'
})
html = urllib.request.urlopen(req, timeout=20).read().decode('utf-8', 'replace')
```

掘金文章通常是 SSR / Nuxt 页面，HTML 里可能直接包含 `mark_content:"..."`。如果页面源码里有 `__NUXT__`、`article_info` 或 `mark_content`，优先从 `mark_content` 提取正文，而不是用浏览器肉眼复制或凭标题猜测：

```python
import re, json
m = re.search(r'mark_content:"((?:\\\\.|[^"\\\\])*)"', html, re.S)
if m:
    markdown = json.loads('"' + m.group(1) + '"')
```

提取后要把 Markdown 图片、链接、代码块、列表符号和引用块清理掉，再按“原味改写”原则转成自然口播段落。详见 `references/juejin-mark-content-extraction.md`。

微信公众号正文常在 `id="js_content"` 的节点中。提取时：

- 先取标题，例如 `og:title`。
